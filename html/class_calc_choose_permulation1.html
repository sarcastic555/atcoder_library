Detailed Description

固定したn、k<=Kを満たす任意のkに対して二項係数(nCk)、順列(nPk)の計算

コンストラクタ計算量O(K)、取得クエリ計算量O(1)
10000_C/P_0, 10000_C/P_1, ... 10000_C/P_100までを前計算したい場合
CalcChoosePermulation1 ccp(10000, 100);
ccp.GetChoose(2); // 10000_C_2を返す
ccp.GetPermutation(2); // 10000_P_2を返す
