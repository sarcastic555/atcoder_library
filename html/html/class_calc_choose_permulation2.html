Detailed Description

n、k <= Nを満たす任意のkに対して二項係数(nCk)、順列(nPk)の計算

コンストラクタ計算量O(N)、取得クエリ計算量O(1)
CalcChoosePermulation2 ccp(10000);
ccp.GetChoose(100, 2); // 1000_C_2を返す
ccp.GetPermutation(100, 2); // 1000_P_2を返す
