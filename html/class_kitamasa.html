Detailed Description

template<typename T>
class Kitamasa< T >

隣接k項間の漸化式と初期値が与えられた時、第n項の値をKitamasa法により求める

a[n] = c[0]*a[n-k] + c[1]*a[n-k+1] ... + c[k-2]*a[n-2] + c[k-1]a[n-1] という漸化式が与えられている時、 a[n] = d[0]*a[0] + d[1]*a[1] ... + d[k]*a[k] と表せるベクトルdを出力する
