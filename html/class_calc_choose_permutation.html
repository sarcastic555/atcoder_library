Detailed Description

ある整数Nをコンストラクタで指定し、n<=N、k<=Nを満たす任意のm, kに対して二項係数(nCk)、順列(nPk)をO()計算する

コンストラクタ計算量O(N)、取得クエリ計算量O(1)
CalcChoosePermutation ccp(10004);
ccp.GetChoose(10000, 2); // 10000_C_2を返す
ccp.GetPermutation(10000, 2); // 10000_P_2を返す
