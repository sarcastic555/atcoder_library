Detailed Description

与えられた木に対して前計算をすることで、最小共通祖先取得クエリを高速で返すクラス

LeastCommonAnscester g(N);
for (int i=0; i<N; i++) {
g.AddEdge(u[i], v[i]); //無向辺として登録されるので、逆辺を登録する必要はない
}
g.Setup(); // 前計算にO(NlogN)かかる
int lca = g.GetLCA(x, y); // LCA取得はO(logN)
